HUGGINGFACE AUTHENTICATION GUIDE FOR EMBEDDINGGEMMA
================================================================================

PROBLEM:
You tried to load EmbeddingGemma-300M but got an error:
"You are trying to access a gated repo."

This model requires HuggingFace authentication.

================================================================================
SOLUTION: 3-STEP PROCESS
================================================================================

STEP 1: REQUEST ACCESS (1 minute)
----------------------------------
1. Open your browser and go to:
   https://huggingface.co/google/embeddinggemma-300m

2. Click the button "Agree and access repository"
   (You may need to create a HuggingFace account first)

3. Access is usually granted instantly


STEP 2: GET YOUR TOKEN (1 minute)
----------------------------------
1. Go to: https://huggingface.co/settings/tokens

2. Click "Create new token"
   - Name: "notebook-access" (or any name)
   - Type: "Read" (default)

3. Click "Create token"

4. COPY the token (starts with "hf_")
   Important: You won't see it again!


STEP 3: AUTHENTICATE IN NOTEBOOK (30 seconds)
----------------------------------------------
1. Open evaluate_model.ipynb

2. Find the NEW section "6a. HuggingFace Authentication"

3. In that cell, uncomment ONE of these options:

   Option A (RECOMMENDED - Interactive):
   ```python
   login()
   ```

   Option B (Direct token):
   ```python
   login(token="hf_xxxxxxxxxxxxx")  # Paste your token here
   ```

4. Run the cell and paste your token when prompted

5. Re-run cell 6b to evaluate all models including EmbeddingGemma
